# Carry Pack Logistics Website

A professional, modern Next.js website for Carry Pack Logistics - your trusted logistics partner in Ahmedabad.

## 🚀 Features

- **Modern Design**: Clean, responsive, and professional interface
- **Fast Performance**: Built with Next.js 16 and optimized for speed
- **Premium Animations**: Smooth transitions and loading states
- **Mobile-First**: Fully responsive across all devices
- **SEO Optimized**: Proper metadata and semantic HTML
- **User-Friendly**: Intuitive navigation and clear CTAs

## 📦 Services

The website showcases all our logistics services:

1. **Separate Vehicle Service** - Dedicated vehicles for exclusive cargo
2. **Sharing Vehicle Service** - Cost-effective shared transportation
3. **Warehouse Solutions** - Secure storage with 24/7 monitoring
4. **Car Transport** - Safe vehicle transportation
5. **Bike Transport** - Expert motorcycle handling
6. **Door-to-Door Service** - Complete end-to-end logistics

## 🚦 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📱 Build for Production

```bash
npm run build
npm start
```
